# v1.0.0+21

> 发布日期：2026-07-08
> 前一构建：v1.0.0+20
> 状态：🟢 已发布（TestFlight 上传成功，等待 Apple 端处理 5–15 分钟）

> 本版核心主题：**登录方式扩展** —— 登录页新增手机号验证码登录与 Google 登录完整链路。

## ✨ 新增功能

- **手机号验证码登录**：登录页新增「使用手机号登录」入口，支持区号选择（搜索 / 自定义 / 默认 +86）、获取验证码 60s 倒计时、验证码登录，与 Apple / Google 登录复用同一状态机和 `needsUsernameSetup` 兜底（`2e6d6fc`）
- **Google 登录客户端完整接入**：基于 `google_sign_in` 7.x 方式 B，登录页 Google 按钮接通完整链路（拿 idToken → POST `auth/google/login` → 复用 `LoginResponse`），凭据走 Info.plist URL scheme + Dart 参数，不放 `GoogleService-Info.plist` 进工程（`1350445`）
- **Google 登录请求体对齐后端验签流**：由 `{id_token}` 改为 `{code: idToken}`，由后端直接验签 idToken（非授权码换取）；dev 日志改为只打印 idToken 长度 / 前缀，不再泄露完整 JWT（`2e6d6fc`）
- **登录页文案微调**：Apple / Google 按钮文案由「继续」改为「登录」，短信登录全套文案入 l10n（`2e6d6fc`）
