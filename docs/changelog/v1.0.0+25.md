## ✨ 新增功能

- **截屏 Bug 反馈（内部测试）**：在应用内截屏后自动弹出反馈表单，填写 Bug 描述并提交，工单（含截图 + 描述 + 设备 / 版本 / 用户 / 页面信息）自动上报到团队 GitHub 工单仓库 `PoeticBear/app-bug-reports`。测试同学反馈问题不再需要切到别的工具截屏 + 打字 + 找人转发，截屏即上报。（`218f9c1` `7d0ee57`）

## 🔧 构建 / CI

- **TestFlight 隔离**：截屏反馈功能仅在 Debug / TestFlight 构建启用；App Store 正式上架包通过编译期 `--dart-define=FEEDBACK_ENABLED` + 运行时 `sandboxReceipt` 双保险**物理剔除**（tree-shake），线上用户完全无感、也接触不到内嵌的工单 token。（`218f9c1`）
