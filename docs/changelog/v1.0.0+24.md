## ✨ 新增功能

- **底部「+」Tab 弹 Popup 菜单分流发帖入口**（`b8baae6`）— 选「写文字」push `TextNotePage`（与 +23 一样走截图交接 ComposePost），选「普通图文」切到 `tab=2` 进入 `ComposePost`，原有草稿拦截保留。

## 🐛 修复

- **登录页加载动画 bug：两个按钮同时转圈**（`b3e8d29`）— 账号密码 / Apple 按钮内的 spinner 三元删除，恢复原文本/图标；新增私有 `_loadingOverlay` helper 用 `Stack + Positioned.fill + IgnorePointer + 半透明黑 + 居中 spinner` 覆盖全页；顶层套 `PopScope(canPop: !_isLoading)` 拦截转圈期间系统返回手势，避免状态泄漏。
- **定时发布字段读取错位**（`cb9884d`）— 服务端 openapi 契约字段是 `scheduled_publish_time`，客户端历史代码读的是 `scheduled_time` / `scheduledTime`，导致 `PostModel.scheduledTime` 落空。改为 `scheduled_publish_time → scheduled_time → scheduledTime` 三档兜底，`PostService.createPost` / `getScheduledPosts` / `PostState.createPost` 同步打印入参与服务端响应便于核对。
- **底部导航第 4 项未读红点漂在 Tab 右上角**（`b8baae6`）— `Positioned(right: 22, top: 14)` 让 8×8 红点挂在心形 icon 右上肩附近，与 icon 轮廓轻微重叠。
- **红点色系不统一**（`b8baae6`）— 底部导航红点 + 通知列表项未读指示器 同步从 `appColors.accent`（蓝）改为 `appColors.destructive`（红），与消息列表未读数字 badge 形成统一「未读/提醒」色系。

## ♻️ 重构

- **搜索历史从 Dismissible 列表改为 chip 阵列**（`947a5f9`）— `Wrap` 横向排列，点击直接复用 query 触发搜索；移除 `_historyTypeIcon` / `_formatTime` / `_buildHistoryItem` 旧实现，删 `services/search_service.dart` 的无引用 import。
