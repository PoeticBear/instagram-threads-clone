## ✨ 新增功能

- **聊天 / 私信「使用协议」拦截**：进入消息页前先弹出底部协议 sheet，必须同意「最终用户许可协议」全文后才会加载会话列表；点「不同意」或关闭会退回信息流且**不记录**同意状态（下次仍弹），下拉 / 点遮罩均无法绕过。同意状态按协议版本持久化，协议改版后会强制重新同意。（`00dfa06`）
- **注销账号入口**：个人中心新增「删除账号」菜单项，进入独立注销页走完确认流程（满足 App Store 审核指南 5.1.1(v)）。后端 `DELETE /user/me` 待就绪，当前先做本地登录态清理兜底，失败时保留登录态可重试。（`00dfa06`）

## 🎨 样式调整

- **Apple 登录按钮改用官方组件**：登录页从自绘按钮换为 `SignInWithAppleButton`，满足 HIG（官方 Apple 字形、本地化标题「通过 Apple 登录 / Continue with Apple」、点击反馈、无障碍语义）；深色主题下改白底黑字避免融入背景，并与下方 Google 按钮的高度 / 字号对齐。（`00dfa06`）
