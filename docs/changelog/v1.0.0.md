# v1.0.0

> 发布日期：2026-06-22
> 涵盖构建：+10 ~ +17（共 8 次 TestFlight 发布）
> 状态：🟢 当前版本
> 配套客户端版本：`client/pubspec.yaml` → `version: 1.0.0+17`

v1.0.0 是项目在 iOS TestFlight 上的**首发版**。本节汇总从 `+10` 首次上传到 `+17` 当前构建期间所有面向用户可见的功能新增、问题修复、性能/样式优化、架构重构、文档与构建工具变更。

> 单构建细目见 git log；本文件按 SemVer 总览视角组织。

---

## ✨ 新增功能

- **首页底部导航栏毛玻璃样式** (`2068d8e`) — 跟随后台主题色 + 高斯模糊，提升视觉通透感与原生感
- **个人中心新增「转发」Tab** (`0d22d34`) — 与帖子、回复 Tab 并列，展示当前用户转发的内容
- **iOS 首启网络权限弹窗静默** (`e9979ac`) — 启动期不再突兀弹起网络权限对话框

## 🐛 修复

- **ITMS-90683 修复** (`156ab67`) — 补齐 `NSLocationAlwaysAndWhenInUseUsageDescription`，过审 App Store
- **ApiClient 401 自动 refresh + 全局登出兜底** (`c0638c0`) — Token 过期时自动续签，续签失败统一跳回登录
- **头像/昵称更新后核心三页实时刷新** (`fa3a990`) — 全局订阅 `AuthState`，改资料后 Feed / Profile / ComposePost 立即生效
- **Feed 流视频静音改为全局共享** (`fe3b29a`) — 视频池内统一开关，避免多视频独立静音状态错乱
- **编辑资料表单同步 `username` 字段** (`4e279b3`) — 修复 `username` 缺失同步导致保存后不一致
- **首页快捷发帖区头像 fit 改为 cover** (`b989e62`) — 避免竖屏图片在圆形头像框内拉伸变形
- **发帖页顶部用户名称空白兜底** (`b3f949c`) — 异步加载期间显示 `username` 兜底，不再空白
- **个人中心首登名称空白修复** (`e9979ac`) — 新会话首次进入个人中心时回退显示 `username`

## ♻️ 重构

- **统一网络错误处理** (`0d22d34`) — ApiClient / Service 层错误码归一化，页面只需关心业务分支
- **发帖页工具栏适配** (`0d22d34`) — 顶部/底部工具栏重新排布以配合统一错误处理后的交互

## 🎨 样式调整

- **首页底部导航栏毛玻璃参数微调** (`4e11a78`) — 增强通透感，进一步贴近 iOS 原生玻璃质感

## 📝 文档

- **CLAUDE.md 新增「一键发布到 TestFlight」工作流** (`b1cf93d`) — 收录 `client/scripts/release.sh` 触发指令与失败处理
- **新增首页帖子信息流代码定位文档** (`4f29f53`) — `docs/code-locations/feed-page.md` / `feed-post-widget.md`

## 🔧 构建 / CI

- **新增 `client/scripts/release.sh` 一键发布脚本** (`87521ba`) — 实时进度展示 + 日志落盘 + 卡顿检测 + 自动重试，支持 `--only-upload` / `--no-bump` / `--no-push` 参数
