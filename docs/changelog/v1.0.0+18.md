## 🐛 修复
- 修复引用卡头像长图（竖屏 / 横图）变形：之前 `CachedNetworkImage` 缺 `fit` 参数，默认 `BoxFit.fill` 会把图片横向拉伸成"胖"圆形。改为 `BoxFit.cover` 后等比缩放居中裁切，与主帖头像（`UserAvatarWithFollow`）行为一致
- 顺带修复帖子回复列表头像的同源 Bug：`_buildAvatar` 内 `CachedNetworkImage` 同样缺 `fit`，现统一改为 `BoxFit.cover`（涉及 `feedpost.dart:158` 与 `reply_bottom_sheet.dart:383`）
- 增强引用卡 / 回复列表头像的加载与失败态：之前无 `placeholder` / `errorWidget`，加载中或失败时头像区域为空白。现补齐两个回调——加载中显示居中 spinner，失败时降级为圆形默认头像（`Icons.person`），体验与首页快捷发帖区（`feed.dart`）一致
