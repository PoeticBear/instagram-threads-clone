# 变更日志索引

本目录按版本号组织每次发版的更新记录，命名规则：

- **摘要文件**（可选）：`v{主}.{次}.{修}.md`（如 `v1.0.0.md`）—— 用于 SemVer 升级时对该版本做总览
- **构建文件**（必填）：`v{主}.{次}.{修}+{构建号}.md`（如 `v1.0.0+18.md`）—— 每次 TestFlight 发布后新建

每次新增构建文件后，请把对应行追加到本文件「全部版本」表的**最上方**。

> 模板见 [`template.md`](./template.md)。
