# Tweet 官网产品需求文档（PRD）

> 用途：交付给前端开发，用于实现 Tweet 官网
> 文档版本：v0.2 · 2026-07-14

---

## 1. 产品愿景

Tweet 是一款 iOS 社交 App，定位「对话优先」—— 没有算法决定你看什么，只有你选择关注的人。文字优先，图片友好，隐私优先，为 iPhone 原生打造。

**官网是这个理念的对外窗口**：让第一次听说 Tweet 的人 30 秒内明白它是什么、为什么用、在哪里下载；让老用户能查到隐私政策、联系到开发者。

---

## 2. 目标用户

| 用户类型 | 来官网做什么 | 关注点 |
|---|---|---|
| 潜在下载用户 | 从 App Store / 媒体渠道点进来 | 是什么 / 截图 / 能不能信任 |
| 老用户 | 查隐私政策、找客服 | 隐私条款 / 联系方式 |
| 媒体 / KOL | 写报道或推荐 | 产品故事 / 核心功能 / 截图 |
| App Store 审核员 | 验证 `marketing_url` / `support_url` / `privacy_url` | 三个 URL 能打开、内容完整 |

---

## 3. 调性与文案风格

### 3.1 调性主轴

- **副标题**：Share what matters, right now.
- **关键词**：social, threads, microblog, posts, chat, network, follow, feed, share, community
- **核心差异点**（沟通主轴）：

  1. **无算法时间线** —— 按时间顺序，只看你选择关注的人
  2. **文字优先，图片友好** —— 280 字 + 图片 + 位置标签
  3. **隐私优先** —— 联系人数据保留在本机，不上传
  4. **为 iPhone 原生打造** —— 干净、流畅的 iOS 体验

### 3.2 双语约定

- **默认语言**：简体中文
- **备选语言**：英文
- **切换器**：每个页面顶部右上角 EN / 中
- **法律页**（隐私政策 / 服务条款）：中英版本独立撰写，不做机器翻译

### 3.3 文案单一信源

所有产品文案以 `client/fastlane/metadata/{en-US,zh-Hans}/` 下文件为锚点，官网不另造文案，App 改了 → 官网同步改。

## 5. 各页面内容

### 5.1 首页 `/`

**目标**：访客 3 秒内明白 Tweet 是什么、为什么用、在哪里下载。

**版块顺序**（自上而下）：

1. **导航栏**：Logo + 5 个导航项 + 语言切换 + App Store 下载按钮
2. **Hero 区**
   - 主标题（zh）：把你最在意的想法，第一时间说出来
   - 副标题（zh）：Tweet 是一款对话优先的 iOS 社交 App —— 没有算法，只有你选择关注的人
   - CTA：App Store 下载按钮
   - 视觉：右侧手机模型 + App 截图
3. **价值三连点**（突出差异点）
   - 无算法时间线
   - 文字优先
   - 隐私优先
4. **核心功能 6 卡**（点击跳 `/features` 详情）
   - 文字 + 图片 + 位置贴文
   - 实时通知
   - 相机拍摄
   - 私聊与群聊
   - 社区与话题
   - 多种登录方式
5. **应用截图轮播**：5-7 张真机截图横向滚动
6. **「为什么选择 Tweet」**（差异点展开）
   - 公开对话
   - 为 iPhone 而生
   - 你掌控你的关注
7. **最终 CTA**：准备好加入了吗？→ App Store
8. **页脚**

### 5.2 功能介绍 `/features`

**结构**：9 节，每节包含「功能标题 + 一段描述 + 1-2 张截图 + 3 个 bullet 卖点」。

| # | 功能 | 一句话 |
|---|---|---|
| 1 | 文字 + 图片 + 位置贴文 | 280 字以内 + 图片 + 可选位置标签 |
| 2 | 时间线动态（无算法） | 按时间顺序，只看你关注的人 |
| 3 | 实时通知 | 点赞、回复、@、关注的实时提醒 |
| 4 | 相机集成 | 前后摄、闪光、超广角、缩放 |
| 5 | 私聊与群聊 | 1v1 + 群组消息，支持表情回应 |
| 6 | 社区 | 加入兴趣社区，与同好互动 |
| 7 | 话题 | 围绕话题聚合内容 |
| 8 | @提及与引用 | @提醒、引用转发、对话串联 |
| 9 | 多种登录方式 | Apple / Google / 手机号 / 用户名 |
